# Plans

This directory stores project plans, proposals, and task breakdowns created by the assistant.

Conventions
- Location: repository root `plans/`
- File format: Markdown
- Naming: `YYYYMMDD-short-title.md`
- Frontmatter (recommended): `title`, `author`, `date`, `status`

Workflow
- The assistant creates new plan files here when you ask it to implement or propose work.
- Files are created in the working tree; the assistant will ask before creating commits or PRs unless configured otherwise.

If you want automatic commits or PRs, tell the assistant to enable that in preferences.
